## 🚀 Features

- Simulates quantum systems with up to **10 qubits**
- Supports basic gates: `X`, `Y`, `Z`, `H`, `S`, `T`, `CNOT`
- Implements **Grover’s** and **Shor’s** algorithms
- Built-in **GUI with visualization**
- Perfect for **educational purposes**
